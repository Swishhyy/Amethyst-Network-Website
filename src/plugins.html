Server Plugins

Learn about the plugins that enhance your gameplay experience

Amethyst Network runs a carefully selected set of plugins that enhance the vanilla Minecraft experience while preserving its authentic feel. These plugins add quality-of-life improvements, protection systems, and new gameplay elements.

AuraSkills

RPG Leveling

A comprehensive RPG progression system that lets you level up various skills like mining, farming, and combat to unlock special abilities and perks.

Basic Commands

  • /skills - View your skills and levels
  • /skills top - View the server's top players for each skill
  • /skill [skillname] - View detailed information about a specific skill
  • /stats - View your player stats that are affected by your skills
  • /mana - Check your current mana level (used for abilities)

How to Level Up

  1. Each skill can be leveled by performing related activities
  2. Mining skill increases when mining stone and ores
  3. Combat skills increase when fighting mobs
  4. Farming increases when harvesting crops
  5. Check /skills menu for all available skills and how to level them

Towny

Land Claiming

Create and join towns with other players, claim territory, and build communities with advanced protection and permissions systems.

Basic Commands

  • /town new [townname] - Create a new town
  • /town claim - Claim the chunk you're standing in for your town
  • /town add [player] - Invite a player to your town
  • /town join [townname] - Join a town that invited you
  • /plot fs [amount] - Put your plot up for sale
  • /t spawn - Teleport to your town's spawn point

Getting Started with Towny

  1. Earn enough money to start a town (use /bal to check balance)
  2. Use /town new [name] to create your town
  3. Use /town claim to claim the land around you
  4. Invite friends with /town add [player]
  5. Set up plots with /plot set [type] (shop, arena, etc.)

MythicMobs

Custom Mobs & Boss Fights

Enhances your combat experience with custom mobs, epic boss fights, and special loot drops that provide unique challenges and rewards.

Features

  • Custom bosses with unique abilities and attack patterns
  • Special mobs with enhanced AI and behavior
  • Rare loot drops with custom enchantments
  • Scheduled boss events with server-wide notifications
  • Progressive difficulty scaling with player gear

Boss Fight Tips

  1. Boss spawn locations are marked on the server map
  2. Bosses have special abilities - watch for visual cues
  3. Some bosses require specific strategies or items
  4. Team up with other players for tougher bosses
  5. Higher difficulty bosses drop better loot

ExecutableItems

Custom Items

Discover and craft unique items with special abilities, custom textures, and powerful effects that can be found throughout the world.

Features

  • Custom weapons with unique abilities
  • Special armor sets with set bonuses
  • Magical tools that enhance resource gathering
  • Custom textures for unique visual appearances
  • Items with active and passive abilities

How to Use Custom Items

  1. Custom items can be found in dungeon chests, boss drops, or crafted
  2. Right-click to activate special abilities (if available)
  3. Some items have passive effects that are always active
  4. Certain items may require mana or cooldowns for abilities
  5. Check item lore for information on how to use each item

CoreProtect

Protection

Track and restore block changes in case of grief or accidents. Our staff uses this tool to investigate and revert unwanted changes.

Staff Commands

While regular players cannot use these commands, you can report incidents to staff who will use CoreProtect to:

  • Check who placed or broke blocks in an area
  • Restore modified terrain after grief
  • Track item transactions and inventory changes

Vault & Economy

Economy

Our server features a player-driven economy where you can earn, spend, and trade Amethyst Crystals (our server currency).

Basic Commands

  • /balance - Check your current balance
  • /pay [player] [amount] - Send money to another player
  • /baltop - View the richest players on the server

How to Earn Money

  • Selling items to other players
  • Completing server quests and challenges
  • Participating in server events
  • Finding special amethyst crystal ores in caves

BlueMap

Utility

BlueMap provides a beautiful 3D map of our server world that you can view in your browser. Explore the terrain, locate biomes, and find your way around.

How to Use BlueMap

  1. Visit our BlueMap in your browser
  2. Navigate the map using your mouse to pan and zoom
  3. Toggle different map layers using the controls in the top-right
  4. Find players and points of interest on the map

EssentialsX

Utility

EssentialsX provides a variety of quality-of-life commands and features that enhance your gameplay experience.

Useful Commands

  • /home - Teleport to your home location
  • /sethome [name] - Set a home location
  • /tpa [player] - Request to teleport to another player
  • /spawn - Return to the server spawn point
  • /msg [player] [message] - Send a private message
  • /afk - Mark yourself as away from keyboard

AmethystEnhancements

Fun

Our custom plugin that adds special abilities and features related to amethyst crystals throughout the world.

Custom Features

  • Enhanced amethyst crystals with special properties when mined
  • Amethyst crystal caves with unique mob spawns
  • Craftable amethyst tools and armor with special enchantments
  • Rare amethyst geode structures with valuable loot

Amethyst Crystal Types

Regular Amethyst Used for basic crafting and currency
Charged Amethyst Provides night vision when held
Resonant Amethyst Emits particles and can be used in special recipes
Pure Amethyst Extremely rare, used for the most powerful items

Simple Voice Chat

Communication

Talk with nearby players using real-time voice chat, creating a more immersive multiplayer experience.

Features

  • Proximity-based voice chat (players can only hear others who are nearby)
  • Adjustable voice chat range
  • Push-to-talk or voice activation options
  • Group chat channels for teams and towns
  • Compatible with the Simple Voice Chat mod

How to Use Voice Chat

  1. Install the Simple Voice Chat mod for your client
  2. Connect to our server as normal
  3. Press V (default key) to open the voice chat settings
  4. Configure your microphone and other preferences
  5. Press the configured key (default: CAPS LOCK) to talk

Citizens

NPCs

Interact with custom NPCs throughout the world that serve as quest givers, merchants, and guides for server features.

Features

  • Interactive NPCs with dialogue options
  • Quest NPCs that give and track progress on server quests
  • Merchant NPCs with special shops and unique items
  • Guide NPCs that provide information about server features
  • NPCs with unique appearances and skins

How to Interact with NPCs

  1. Approach an NPC in the world
  2. Right-click on the NPC to start a conversation
  3. Click on dialogue options to navigate conversations
  4. Some NPCs may offer quests, shops, or other interactions
  5. Look for NPCs in towns, at spawn, and other important locations
